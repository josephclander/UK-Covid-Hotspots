Message about the Data

Data is being compared from current information on cumulative cases in local authorities in the UK. Cumulative cases alone do not take into account the size of the local population or the change in the number of cases.

To get a better feel for how cases are changing for local authority areas, a comparison has been added on new cases compared to each area's previous cummulative cases.

Data is collected from coronavirus.data.gov.uk
